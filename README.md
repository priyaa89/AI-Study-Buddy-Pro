# 📚 AI Study Buddy Pro

## Overview

AI Study Buddy Pro is an interactive educational platform designed to help students improve productivity, learning efficiency, and study organization. The application combines multiple study tools within a modern and attractive user interface.

## Features

* 📄 Notes Summarizer
* 🧠 Flashcard Generator
* ✅ Quiz Generator
* 📅 Study Planner
* ⏱ Pomodoro Timer
* 🔥 Motivation Center
* 📊 Progress Dashboard
* 🎨 Modern Glassmorphism UI

## Technology Stack

### Frontend

* Python

## Project Architecture

User → Dashboard → Learning Modules → Results Display

Modules:

* Notes Summarizer

* Quiz Generator
* Study Planner

* Motivation Center

## Installation

Clone the repository:

git clone https://github.com/yourusername/AI-Study-Buddy-Pro.git

Install dependencies:

pip install -r requirements.txt

Run the application:

python -m streamlit run app.py
